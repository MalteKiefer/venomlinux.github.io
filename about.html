Venom Linux is a source based linux distribution originated in Malaysia comes with multilib support targetting experienced users. This distro is inspired by CRUX of its simplicity and its use 'KISS' philosophy. Which is use simple tar compressed package, ports system for packages, BSD-style initscripts and a few collection of small packages which is trimmed down by removed unused stuff like locale, doc, gtk-doc, info pages (man pages is kept) and etc. Venom Linux use custom package manager which fully written in bash for installing, upgrading, removing and etc.

Venom Linux features:

  • Package build script is BSD's ports style
  • Using BSD-style init
  • Easy to write your own ports
  • Multilib support
  • Slim, packages is trimmed by removing all unused stuff
