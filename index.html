Venom Linux is a source-based Linux® distribution originally built on top of LFS (Linux From Scratch) then independently grown to make it suitable for daily use. This distro is inspired by CRUX because of its simplicity and KISS philosophy.

Venom Linux uses BSD-style initscripts and a BSD-like ports system for packages. Packages are managed by its own package manager, called scratchpkg, that has dependency checking and is fully written in Bash.
